Projeto acadêmico

Jogo da Forca – Aplicativo em C# com Windows Forms

Desenvolvido como parte de um trabalho acadêmico, este projeto consiste em um jogo da forca interativo criado com C# utilizando Windows Forms (.NET Framework). O objetivo é adivinhar uma palavra oculta a partir de uma dica ou tema, revelando progressivamente as letras corretas. O sistema contabiliza erros, limita tentativas e exibe graficamente o avanço do jogador. A interface é intuitiva e proporciona uma experiência divertida e educativa, reforçando conceitos de lógica de programação, manipulação de strings e interação com o usuário em aplicações desktop.
